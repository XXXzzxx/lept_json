## '\u'解析
1. 先检测Unicode格式 \u 后加4个十六进制数
2. 代码中每次使u*16  `*u = *u << 4; `  使用u计算unicode(字符转数字)值, 后面还使用了或(|)操作其实也就是相当于是加操作
3. 如果格式正确进行判断是否有低代理项, 如果有检查格式正确后, 使用`u = 0x10000 + (uh - 0xD800) * 0x400 + (ul - 0xDC00);`  计算码点
4. 最后把码点编码成utf-8
